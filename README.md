Para verificar que algunas partes del programa funcionen correctamente, se realizaron pruebas unitarias utilizando la librería unittest de Python. Estas pruebas permiten comprobar que ciertas funciones del código trabajen como se espera de manera independiente. En este caso se probaron las funciones validar_fecha y validar_email, revisando tanto casos válidos como inválidos para asegurarse de que el sistema identifique correctamente los datos incorrectos. Las pruebas se aplicaron al código desarrollado durante el laboratorio, con el objetivo de comprobar que las validaciones implementadas funcionen adecuadamente y que el programa responda de forma correcta ante diferentes tipos de entrada.
